ADVENTURE FIENDS
WEB GAME USING JS WITH PHASER 3

An arcade style game which is heavily inspired by bubble bobble with my own twist. It was created as a project for my studies with the help of the Phaser 3 library in JavaScript. You can play with up to 2 players on one system and beat waves of different enemies through out progressively harder levels, collect score and power ups and ultimately defeat bosses to reach the end.
